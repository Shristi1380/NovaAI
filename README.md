# 🤖 NovaAI

NovaAI is a full-stack AI chatbot that provides real-time conversational responses using Google's Gemini API. It features a modern chat interface, persistent chat history, and a scalable backend.

## ✨ Features

- AI-powered conversations using Gemini API
- Real-time chat interface
- Chat history stored in MongoDB
- Responsive UI
- REST API architecture

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Google Gemini API

## 📂 Project Structure

```
NovaAI
│
├── Frontend
└── Backend
```

## 🚀 Run Locally

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm start
```
